Jeu de Memory

Le jeu du memory (ou jeu de paires) est un jeu de société édité pour la première fois par l'entreprise Ravensburger en 1959. Il se joue de 2 à 8 joueurs. Depuis, il a été décliné en de nombreuses versions mais le principe reste le même. Une variante possible consiste en l'introduction d'une carte sans appariement, qui reste donc la dernière carte en jeu après que toutes les paires ont été découvertes. Le jeu a été décliné en de nombreux thèmes (publicitaires, thématiques, à licences). Certaines éditions sont désormais épuisées et remplacées par de nouvelles aux thèmes plus modernes. L'éditeur propose aussi à ses clients de créer leur propre jeu, avec leurs propres photos


Règles du Jeu

Le jeu se compose de paires de cartes portant des illustrations identiques. A chaque tour les joueurs doivent retourner 2 cartes : si les images affichées sont identiques alors le joueur peut continuer son tour, sinon les cartes sont retournées face cachée et tour passe au joueur suivant.
Lorsque toutes les cartes ont été retournées, le joueur qui a reformé le plus de paires est déclaré vainqueur.
En mode 1 joueur, l'objectif du jeu est de retouver toutes les paires en le moins de coups possible. Votre score sera le nombre de coups dont vous avez eu besoin.
